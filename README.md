# journal.py

This script provides a simple command-line journal manager that saves entries to `journal.txt`.

## Features

- Add a dated journal entry
- View all saved entries
- Search entries by keyword
- Delete all entries after confirmation

## Requirements

- Python 3.x

## Usage

Run the script from the repository folder:

```bash
python journal.py
```

Choose one of the menu options:

1. Add Entry
2. View Entries
3. Search Entries
4. Delete all entries
5. Exit

## Notes

- Journal entries are stored in `journal.txt` in the same folder.
- The file is created automatically when the first entry is added.
- Deleting all entries clears the file contents.
